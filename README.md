# TaskSync

Synchronizes Tasks between different Task and Issue management systems.

## Installation

Add this line to your application's Gemfile:

    gem 'tasksync'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install tasksync

## Usage

    $ tasksync --help
